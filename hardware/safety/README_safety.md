# Safety documentation

This hardware includes a high-energy battery, a rotating chain drive, and high-current electrical paths. Follow all checklist items before operation.

Minimum requirements:

- Chain guard installed.
- 30 A fuse installed in battery-to-ESC positive line.
- Emergency stop verified.
- Battery inspected for swelling, damage, and loose terminals.
- ESC and wiring insulated.
- Two-person rule used for initial testing.
- Class C fire extinguisher available.
