# Mechanical assembly

This folder contains mechanical guidance for frame construction, shaft/transmission alignment, torque arm fabrication, and floating-caliper torque measurement.

## Critical dimensions

| Component | Dimension | Tolerance |
|---|---:|---:|
| Base frame | 600 x 400 mm | +/- 2 mm |
| Motor mounting plate | 150 x 100 x 5 mm | +/- 1 mm |
| Shaft center distance | 200 mm | +/- 0.5 mm |
| Torque arm length | 100.0 mm | +/- 0.1 mm |
| Torque arm inclination | 40 degrees | +/- 1 degree |
| Brake disc runout | - | < 1 mm |

## Assembly sequence

1. Cut 40 x 40 x 2 mm steel hollow sections to the specified frame dimensions.
2. Tack weld the base frame and verify diagonal alignment.
3. Fully weld the structural frame while maintaining the shaft center distance.
4. Install pillow block bearings and verify free shaft rotation.
5. Mount the BLDC motor and align the chain/sprocket transmission.
6. Install the brake disc and floating-caliper mechanism.
7. Weld or mount the torque arm at 40 degrees.
8. Mount the load cell vertically and verify that the force path is axial.
9. Install the chain guard before operation.

See `assembly_photos/` and `floating_caliper_photos/` for visual guidance.
