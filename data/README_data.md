# Data

This folder provides raw and processed data tables used to support telemetry validation, torque calibration, steady-state characterization, dynamic load-step response, uncertainty analysis, and model-vs-experiment comparison.

- `raw/`: data as acquired or tabulated from the experimental setup.
- `processed/`: processed data used to regenerate figures and tables.

The values are organized to match the revised HardwareX manuscript and can be used with the scripts in `scripts/`.
