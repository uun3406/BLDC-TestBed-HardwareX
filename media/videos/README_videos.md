# Supplementary videos

The supplementary videos are hosted in the Zenodo archive and mirrored as appropriate in the project repository.

Expected video files:

1. `video_01_assembly_overview.mp4` - frame, motor, chain, brake, load cell, ESC, and ESP32 overview.
2. `video_02_torque_calibration.mp4` - zeroing, known mass loading, and verification mass.
3. `video_03_throttle_calibration.mp4` - DAC-side and ESC-side voltage at 0/25/50/75/100% command.
4. `video_04_normal_operation.mp4` - telemetry and torque logging during stable operation.
5. `video_05_load_step_test.mp4` - manual brake load step and dynamic response.
6. `video_06_emergency_stop.mp4` - emergency stop and zero-throttle behavior.

If GitHub file-size limits are exceeded, keep the `.mp4` files in Zenodo and retain this folder as a manifest pointing to the permanent archive.

Zenodo DOI: https://doi.org/10.5281/zenodo.18689715
