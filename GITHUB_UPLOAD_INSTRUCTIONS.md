# GitHub upload instructions

1. Create or open the repository `BLDC-TestBed-HardwareX` on GitHub.
2. Upload all folders and files in this package.
3. Keep the Zenodo archive as the permanent, citable, frozen version.
4. If video files are too large for GitHub, keep the videos in Zenodo and retain `media/videos/video_manifest.csv` plus `media/videos/README_videos.md` in GitHub.
5. After uploading, verify that the following links work from the README:
   - wiring diagram
   - bill of materials
   - firmware
   - raw data
   - processed data
   - plotting scripts
   - video manifest
